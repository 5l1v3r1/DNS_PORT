* set port 80
* run

# Installation
* ``` $ cd ~/.recon-ng ```
* ``` $ cp -r /usr/share/recon-ng/modules/* . ```
* ``` $ cd modules/recon/domains-hosts ```
* ``` paste smart_port.py here ```
* ``` recon-ng ```
* ``` search smart ```
